# Chrome extensions → Firefox

Read from your Chrome **Default** profile
(`~/Library/Application Support/Google/Chrome/Default/Extensions/`). Each Chrome
extension ID was looked up to a name, then checked for a Firefox equivalent.

Firefox is your chosen primary browser. Where there's **no good Firefox add-on**,
it's flagged so you can decide whether to also install Chrome/Brave (which would
let you reinstall the original extension unchanged).

> Tip: install the "available" ones, then revisit the "no equivalent" list
> at the bottom — that's your decide-on-Chrome list.

---

## ✅ Has a Firefox equivalent

- [ ] **1Password – Password Manager** — https://addons.mozilla.org/firefox/addon/1password-x-password-manager/
- [ ] **Vimium** → Vimium-FF — https://addons.mozilla.org/firefox/addon/vimium-ff/
- [ ] **AdBlock** → **uBlock Origin** (recommended; Firefox is its best home) — https://addons.mozilla.org/firefox/addon/ublock-origin/
- [ ] **Video Speed Controller** → Videospeed — https://addons.mozilla.org/firefox/addon/videospeed/
- [ ] **GoFullPage** (full-page screenshot) → **FireShot**, or Firefox's built-in
      full-page screenshot (right-click → "Take Screenshot" → "Save full page") — https://addons.mozilla.org/firefox/addon/fireshot/

## ❌ No good Firefox equivalent — revisit Chrome if you need these

- [ ] **Claude** (Claude for Chrome) — Chrome-only; no Firefox build. **This is
      the strongest reason to keep Chrome/Brave installed.**
- [ ] **Trustpilot: Reviews & Ratings** — no Firefox add-on found.
- [ ] **Fakespot** — Mozilla *owned* Fakespot and **discontinued it (mid-2025)**,
      removing the built-in Review Checker too. No working equivalent; skip.

---

### Summary

The core of your kit maps cleanly to Firefox (1Password, uBlock/AdBlock, Vimium,
video speed, screenshots). The only genuinely Chrome-locked thing you might miss
is the **Claude extension** — if you use it daily, install Brave or Chrome
alongside Firefox (see MIGRATION.md) and reinstall just that one.
